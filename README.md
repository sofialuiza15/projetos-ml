# Projetos de Machine Learning

Repositório com projetos independentes de Machine Learning e Visão Computacional desenvolvidos em Python, utilizando bibliotecas modernas do ecossistema científico.

---

## Projetos

### 1. Conversão de Imagens para Preto e Branco
**Pasta:** `conversao-pb/`

Projeto de Visão Computacional focado na transformação de imagens coloridas em escala de cinza. Explora as técnicas de manipulação de canais de cor (RGB) e os diferentes métodos de conversão.

---

### 2. Previsão com Múltiplos Datasets
**Pasta:** `previsao-datasets/`

Estudo comparativo de modelos de previsão (forecasting) aplicados a 5 datasets distintos, todos carregados via **sktime**. O objetivo é avaliar o comportamento do modelo frente a séries temporais com características diferentes.

---

### 3. Projeto Final — Previsão de Série Temporal
**Pasta:** `projeto-final/`

Projeto completo de forecasting aplicado a um dataset específico. Engloba todas as etapas de machine learning: pré-processamento, treinamento do modelo e avaliação dos resultados.

---

## Como Executar

Você pode abrir qualquer notebook diretamente no Google Colab clicando no arquivo `.ipynb` dentro de cada pasta e selecionando **"Open in Colab"**.

Ou clone o repositório localmente:

```bash
git clone https://github.com/seu-usuario/projetos-ml.git
cd projetos-ml
```
